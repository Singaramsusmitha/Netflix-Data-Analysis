# 🌟 Overview

Netflix hosts thousands of Movies and TV Shows from around the world. Understanding this massive collection can reveal valuable insights about content trends, production patterns, genres, ratings, and audience preferences.

This project performs a complete **Exploratory Data Analysis (EDA)** on the Netflix Titles Dataset using Python to transform raw data into meaningful business insights through visualization and statistical analysis.

---

# 🎯 Objectives

This project aims to:

* 📌 Clean and preprocess raw Netflix data
* 📌 Handle missing and inconsistent values
* 📌 Perform feature engineering
* 📌 Explore content trends using visualizations
* 📌 Answer real-world business questions
* 📌 Generate actionable insights from data

---

# ❓ Business Problem

Streaming platforms continuously expand their content libraries. Without proper analysis, it becomes difficult to answer questions such as:

* Which type of content dominates Netflix?
* Which countries produce the most content?
* Which genres attract the largest audience?
* How has Netflix expanded over the years?
* Which directors and actors appear most frequently?
* What content ratings are most common?

This project answers these questions using data analytics.

# 📈 Key Insights

📌 Movies account for the majority of Netflix's catalog.

📌 The United States is the largest content producer.

📌 Drama and International Movies dominate the platform.

📌 TV-MA is one of the most frequently assigned ratings.

📌 Most movies are between **80–120 minutes** long.

📌 One-season TV Shows are the most common.

📌 Netflix experienced significant content growth during the late 2010s.
